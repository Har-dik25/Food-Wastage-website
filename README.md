# 🌍 EcoEats: Food Waste Management Platform

EcoEats is a beautiful, modern web platform designed to combat food wastage by helping banquets and individuals donate excess food directly to those in need. With a network of over 150+ banquets, we believe nutritious food should go to people, not landfills.

**[🚀 View Live Demo on Netlify](https://foodwastageproject.netlify.app/)**

## ✨ Features
*   **Premium Modern Interface:** A complete design overhaul featuring an elegant "Dark Mode" aesthetic, vibrant emerald green accents, and advanced glassmorphism components.
*   **Responsive Design:** Fully scaled to look perfect on devices of all sizes using CSS Flexbox and Grids.
*   **Donation Portal:** A fully styled form allowing donors to schedule food collection flexibly (`register.html`).
*   **Informative Sections:** Educational material regarding food waste causes, school/college wastage statistics, and the overarching global perspective.
*   **Interactive UX:** Smooth transition animations, floating card hover effects, and distinct form focus loops.

## 🛠️ Technologies Used
*   **Frontend Structure:** HTML5 (Clean, semantic tagging)
*   **Styling & Animations:** CSS3 (Variables, Keyframes, Flexbox, Gradients, and Backdrop-filters)
*   **Typography:** Google Fonts (`Outfit`)
*   **Icons:** Boxicons

## 🚀 Getting Started
This is a static front-end website, meaning no servers or databases are necessary for local preview.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Har-dik25/Food-Wastage-website.git
    ```
2.  Navigate to the directory:
    ```bash
    cd Food-Wastage-website
    ```
3.  Simply open `index.html` in your favorite modern web browser to view the application!

## 🖼️ User Flows
*   **Home:** `index.html` - Explore our mission and statistics.
*   **Get Involved / Donate:** `register.html` - Fill out details for food collection.
*   **About Us:** `about.html` - Learn about the developer team and our direct contact with 150+ banquets.
*   **Contact:** `contact.html` - Get in touch with us.
